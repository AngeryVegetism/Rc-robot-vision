# Rc-robot-vision

🤖 The project is focused on creating a robot that collects biomedical signals from the environment that can be processed, filtered and analysed using neural network made models inorder to combine the different data modalities to come up with a conclusion that maps out the confidence of a human that is detected and whether they are alive or dead depending on the signals collected and also have the ability for object detection to avoid crash detection.

---

## Features

- Real-time object detection
- REST API backend
- Dashboard visualization
- Database storage
- Mobile-friendly interface

---

## Technologies Used:

- Python
- C
- FastAPI
- Uvicorn
- OpenCV
- Ultralytics
